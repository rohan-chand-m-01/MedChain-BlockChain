## About Passkeys

**Passkeys require initial setup:**

1. **First time**: You must create a passkey using Face ID/Touch ID/Windows Hello
2. **After creation**: You can use the passkey to login instantly

**To create a passkey:**
1. Login with Google/Email first (you already did this ✅)
2. Go to Privy settings in the modal
3. Click "Add Passkey"
4. Follow the browser prompt to create passkey with biometric

**Why passkey button might not work:**
- You haven't created a passkey yet
- Your browser doesn't support WebAuthn (need Chrome/Edge/Safari)
- Windows Hello/Face ID/Touch ID not set up on your device
